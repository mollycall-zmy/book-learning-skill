# Scene Trigger Index / 场景触发索引

This index helps the Agent decide which method cards to invoke for a user task.

| 任务类型 | 语义触发条件 | 典型用户表达 | 优先调用方法卡 | 备用方法卡 |
|---|---|---|---|---|
| 方案评估 | 用户要求判断方案是否合理、可行、可信、有风险 | “靠谱吗”“有没有问题”“帮我审一下”“哪里不对” | 示例方法卡 A | 示例方法卡 B |
| 观点判断 | 用户要求判断某段话、观点、媒体说法是否可信 | “他说得对吗”“这是不是忽悠”“有没有立场” | 示例方法卡 A | 示例方法卡 C |
| 决策建议 | 用户面对多个选择，需要判断该不该做 | “该不该”“选哪个”“值不值得” | 示例方法卡 D | 示例方法卡 B |
| 结构化写作 | 用户需要写方案、汇报、PPT、总结 | “帮我写方案”“帮我搭框架”“这个 PPT 怎么讲” | 示例方法卡 E | 示例方法卡 F |
| 复盘归因 | 用户想知道为什么某件事失败、低效或没有结果 | “为什么没效果”“问题出在哪”“怎么复盘” | 示例方法卡 G | 示例方法卡 B |

## Rules

- 不要只靠关键词匹配。
- 必须包含语义触发条件。
- 必须包含典型用户表达。
- 每个任务类型优先调用 1-2 张方法卡。
- 每次最多调用 3 张方法卡。
- 如果没有合适方法卡，应直接完成任务，并标记未来可补充。
