# Scene Trigger Index / 场景触发索引

| 任务类型 | 语义触发条件 | 典型用户表达 | 优先调用方法卡 | 备用方法卡 |
|---|---|---|---|---|
| 方案评估 | 用户要求判断方案是否合理、可行、可信、有风险 | “靠谱吗”“有没有问题”“帮我审一下”“哪里不对” | 证据与假设检查 | 风险分层判断 |
| 观点判断 | 用户要求判断某段话、观点、媒体说法是否可信 | “他说得对吗”“这是不是忽悠”“有没有立场” | 证据与假设检查 | 立场识别 |
| 决策建议 | 用户面对多个选择，需要判断该不该做 | “该不该”“选哪个”“值不值得” | 风险分层判断 | 资源约束检查 |
| 结构化写作 | 用户需要写方案、汇报、PPT、总结 | “帮我写方案”“帮我搭框架”“这个 PPT 怎么讲” | 结构化表达 | 受众判断 |
| 复盘归因 | 用户想知道为什么某件事失败、低效或没有结果 | “为什么没效果”“问题出在哪”“怎么复盘” | 因果链检查 | 替代解释检查 |

## Rules

- Do not rely only on keywords.
- Match semantic task intent first.
- Call at most 3 method cards.
- If no method card fits, answer directly and mark a possible future card.
